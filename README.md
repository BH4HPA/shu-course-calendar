# ai-schedule-shu-cj

> 小爱课程表 - 上海大学教务管理 (cj.shu.edu.cn) 适配

## 关于

此代码适配上海大学教务管理系统，适配时间2020学年秋季学期，支持上课日期字段，同时支持过往学期。**现已上线。**

## 快速开始

```shell script
$ yarn      # 安装依赖
$ yarn tsc  # 编译typescript
```
